# ChemHub Clean Popularity Edition

Чистая версия ChemHub без накопленного patch-мусора.

## Запуск

```powershell
cd C:\chemhub_popularity_clean_project
docker compose up --build
```

Ссылки:

- Сайт: http://localhost:3000
- AI-агент: http://localhost:5173
- Админ-панель: http://localhost:3100
- API: http://localhost:8000/docs

Админ-пароль по умолчанию находится в `.env`:

```env
ADMIN_PASSWORD=Ch3mHub!Admin#2026
```

## Что включено

- стабильный extractor химических реакций;
- фильтрация мусора, ионных уравнений, катода/анода и электронных балансов;
- условия реакции отдельно от уравнения и визуально над стрелкой;
- сохранение `+`, `↑`, `↓`;
- раскрытие шаблонов `M`, `X`, `Hal`, `Me`, `E` при наличии контекста;
- AI-поиск по химическому смыслу без внешнего API;
- похожие реакции;
- интерактивная таблица Менделеева;
- SEO-страницы реакций, sitemap.xml, robots.txt;
- админ-модерация OCR/сомнительных реакций;
- проверка невозможных реакций эвристиками;
- upsert-логика: если реакция уже есть, сохраняется более подробная версия.

## Важно

Файл `.env` не загружай в публичный GitHub. Используй `.env.example` как шаблон.
